# Service Project Report Template

!!! info "How to Use This Template"
    1. Copy this file and rename it for your project (for example, `2026-Spring-Service-Project.md`)
    2. Fill in each section with the project details
    3. **Privacy**: Use first names, nicknames, or patrol names only - no last names!
    4. Have fun documenting how our troop helped out!

## How to Submit Your Report

There are **two ways** to get your service project report added to the website:

### Option A: GitHub Pull Request (Recommended)

1. Go to our repo: [github.com/Troop-364/T364-web](https://github.com/Troop-364/T364-web)
2. Fork the repository
3. Add your new report file under `docs/service-projects/`
4. Open a pull request when you are done

### Option B: Post on Slack

1. Fill out the template in any text editor
2. Save it as a `.md` file
3. Post it in the troop's Slack channel
4. The webmaster will add it to the website
