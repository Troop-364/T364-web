# Campout Submission Instructions

Use this page if you want to add or update a campout report.

## GitHub Pull Request

Recommended if you are comfortable editing files directly.

1. Go to the troop repo: [github.com/Troop-364/T364-web](https://github.com/Troop-364/T364-web)
2. Fork the repository
3. Add your report under `docs/campouts/364/` or `docs/campouts/5364/`
4. Open a pull request when you are done

## AI-Friendly Workflow

You can use your favorite AI assistant to draft the report before you post it.

1. Copy the campout template into your editor or AI tool
2. Ask the AI to fill in the sections using your notes, photos, and dates
3. Review the draft carefully
4. Make sure names are private and facts are correct
5. Save the finished `.md` file and submit it through GitHub or Slack

### Good AI prompts

- "Turn these campout notes into the Troop 364 report template. Keep it factual and concise."
- "Fill in this Markdown template from these bullet notes, but do not invent details."
- "Rewrite this campout report so it uses first names only and matches the site template."

### Safety checks

- Do not paste private medical, disciplinary, or contact information into an AI tool
- Verify every date, location, and attendee before submitting
- Keep last names out of the report
